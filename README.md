# Notes App
## Мобільний додаток для збереження та зміни нотатків

## Що нового?
- **Світла та темна теми:** тепер у вкладці **Settings** можна легко перемикатися між світлою та темною темою.  
- **Покращена стилізація:** дизайн динамічно змінюється за допомогою хука `useTheme`.  
- **Редагування нотаток:** можливість змінювати текст у вже створених нотатках.  
- **Показ дати:** кожна нотатка відображає дату створення  
- **Сортування нотаток:** за датою або алфавітом для швидшого пошуку.

## Встановлення

1. Клонуйте репозиторій
2. Встановіть залежності: `npm install`
4. Запустіть проєкт: `npx expo start`

5. ## Використані технології

- React Native
- TypeScript
- Expo
